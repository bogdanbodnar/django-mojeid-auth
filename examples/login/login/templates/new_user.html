Create User

Creating users can be automatic. (default behaviour)
Or a custom action can be triggered. (such as redirection to this page)

Note that email (or username or ...) might be already used with another account. The django_mojeid plugin doesn't check for these situations and would simply fail during the user creation. If you want to deal with these situations differently you need to trigger a custom action and put our logic inside.
